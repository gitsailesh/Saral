This folder can be used for uploaded files through application, like doc, xls, pdf, jpg, png, gif etc. 
To store/upload files to this folder you can use $this->getMediaPath() to fetch the path and 
to access files via browser from this folder you can use $this->getMediaURL() to point the location. 
You can create folders under media folder to categories/clasify files uploaded to it. eg: banners, products, companies, resumes, reports, etc
